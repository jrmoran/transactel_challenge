Write the code necessary to find the most common 2-character string.

For example, in the string 'EAEB2D31EFD34ED5RD3EETF' the answer would be 'D3', because it is the most common 2-character string, it appears 3 times.

Just in case your string looks like this '123EEE456' be aware the substring 'EE' is present twice, once at offset 3, and again at offset 4.
